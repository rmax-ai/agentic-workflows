# Iteration History

Store one Markdown file per iteration under year-based folders.

## Layout

- `.agent/iterations/<year>/<timestamp>.md`

Use the loop-provided timestamp path when running through `scripts/run-agentic-workflows-loop.sh`.

## Notes

- Keep each file concise and iteration-scoped.
- Do not append to older iteration files unless correcting a factual error.
- Read only the most recent relevant files when gathering history.
- Legacy migrated entries may use a date-and-slug filename when an exact timestamp is unavailable.
